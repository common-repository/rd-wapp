RD Contacto is a simple plugin so that visitors to your website can make direct contact through WhatsApp, thus creating a direct link between your potential customers, and further personalizing the service to your users.

WhatsApp today has a very wide community worldwide, this plugin facilitates contact with visitors and thus offer a better service and attention.

== WhatsApp button ==

With the WhatsApp chat button, your visitors can send a message directly to your phone number. To start configuring it, you must go to the Tools tab and there you will find the RD WApp option where you can configure the plugin administration data.

== Button shortcode WhatsApp ==

New functionality that allows you to add a Whatsapp button anywhere on our website with just a short code.

example: [rd-btn-whatsapp number="+570000000" text="Whatsapp" message= "Mensaje inicial del chat."]

== Installation ==

1. Go to the Plugins menu in WordPress
2. Search for "RD Contacto"
3. Click "Install"
4. Click "Activate"
5. Go to the WhatsApp chat options in Tools -> RD Contacto
6. Set up contact details
7. Ready to enjoy.

= How to add full phone number in international format? =

Skip the square brackets or hyphens when adding the phone number in international format, just include the country code, area and normal phone number.

Ejemplo: Ej: 580000000000
No use: +58(000)000-0000
